Service Based Email Sendbox

Ideal email type for sharing alerts and announcements

Target is EMAIL Group

Way to create a list for announcements and audit the messaging

Use placeholders to personalize the announcement

Define placeholders in your template to share the announcements. The example is showing a server issue report. However, you may send personalized emails to staff in different departments.

Easy API calls

The service information page gives you a simple JSON body to place in your applications. All you need is to replace the message holder and send it to the API server.
