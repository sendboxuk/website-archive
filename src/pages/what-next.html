What's Next?

The planned development features

 

 

Ignore sensitive values while auditing the emails - COMPLETED

It will be optional to save any values in emails. For example, if you do not want to save a secret value in database, the system will ignore the values and will not save the database. Example: customer Id, serial keys, sensitive information etc...

Online template edit - COMPLETED

Create and edit the email templates using only editor

Send test emails

Online viewing the templates and able to send a test email

Re-populate emails resending from audit log

If the template has been changed or some values are needed to re-submit or missing, you may update the values of placeholders before sending.

Data import

You may upload Excel or CSV file to send the emails.
